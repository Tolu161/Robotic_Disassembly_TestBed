Physical Test Bed 

Robotic Manipulation in Disassembly operations require varying forms of contact rich manipulation tasks from bolt removal, to unplugging for peg-in-hole tasks, snap fit removal. These varying tasks require varied manipulation skills. 
This repository contains the assembly of a physical test bed as shown in the image below. This test bed is modular, multilevel and can be rearranged for varied training or test simulations, it is designed to represent operations encountered in robotic disassembly of EV Batteries.
